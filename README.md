# 简介
一组简单的基于socket的聊天程序，包含独立的三部分。每个文件夹下都是一个独立的聊天程序。
- simple_socket_example/
   - 说明：该目录下是一个客户端文件与服务端文件，共同组成一个简单的1对1通信的聊天程序
   - 运行方法：分别启动`server.py`和`client.py`后，可以在client.py的终端中输入文本，实现客户端与服务端的通信
- chat/ 
  - 说明：一个简单的多人聊天程序，实现了注册、登录、私聊、群聊、私聊的文件传输功能
  - 运行方法：启动`server.py`和若干个`client.py`后，可以注册和登录不同的账户，实现用户间的聊天。
- chat_gui/
  - 说明：基于tkinter库，在`chat/`目录下的程序的基础上实现了简单的图形界面
  - 运行方法同`chat/`
